## 🌐 What This Project Does

This website has two main parts:

### 1️⃣ Landing Page
- Shows all **Projects** with image, title, and description.
- Displays **Happy Clients** with their photos, names, designations, and feedback.
- Contains a **Contact Form** for users to submit their details.
- Has a **Newsletter Section** where users can subscribe via email.

### 2️⃣ Admin Panel
Accessible to admins (no login required in basic version):
- Add/manage **Projects** (including image upload)
- Add/manage **Clients**
- View all **Contact Form submissions**
- View all **Newsletter Subscribers**
